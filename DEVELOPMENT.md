# Процесс разработки
## Task 1
Данное задание заключается в создании _консольного бота_, который просто пересылает полученное сообщение.
Необходимо реализовать заданный функционал с использованием интерфейсов `Reader`, `Writer`, `Handler`.

Статус - <span style="color:green">**Зачтено**</span>

## Task 2
Необходимо реализовать работу _консольного_ и _Telegram_ бота с использованием общих обработчиков.

- Выполнено разбиение на пакеты
- Добавлена возможность лёгкого добавления новых обработчиков
- Каждый бот имеет свой собственный файл запуска. При запуске через главный `Main` файл будет предложен выбор бота
  для запуска

Статус - <span style="color:green">**Зачтено**</span>

## Task 3
- Необходимо реализовать отправку HTTP запросов и получение расписания с сайта УрФУ.
- Необходимо выполнить преобразование полученных данных в сущности кабинетов со всей необходимой информацией.

Статус - <span style="color:orange">**В разработке**</span>

## Task 4

- Реализовать запрос данных с сайта по определённому времени
